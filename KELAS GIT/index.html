belajar git

  • init berfungsi untuk membuat repository baru
  • git status berfungsi untuk mengetahui status file yang ada di repository
  • git add berfungsi untuk menambahkan file ke staging area
  • git commit berfungsi untuk menyimpan perubahan file ke repository
  • git log berfungsi untuk melihat riwayat commit
  • git log --oneline berfungsi untuk
  • git show berfungsi untuk melihat perubahan file
  • git branch berfungsi untuk melihat branch yang ada di repository
  • git checkout berfungsi untuk pindah branch
  • git merge berfungsi untuk menggabungkan branch
  • git clone berfungsi untuk mengcopy repository
  • git pull berfungsi untuk mengambil perubahan dari repository
  • git push berfungsi untuk mengirim perubahan ke repository
  • git remote berfungsi untuk melihat remote repository
  • git fetch berfungsi untuk mengambil perubahan dari remote repository
  • git reset berfungsi untuk menghapus perubahan file
  • git revert berfungsi untuk mengembalikan perubahan file
